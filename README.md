# Kinomore

Front-end приложение для оценки жилой недвижимости с помощью ИИ. Сайт: [[https://kinomore.netlify.com/](https://kinomore.netlify.com/)](https://estate-valuation.tech/)

# Стек технологий:

-   React + React Hooks
-   TypeScript
-   React Query
-   tailwind
-   react-testing-library
-   Formik, Yup
